## 主要内容

需求说明书涵盖：

- **台账数据范围**：关务、财务、税务三方面信息的整合
- **台账主线设计**：以报关单为主线的出口业务台账，以报关单+项号为主线的出口商品台账
- **核心功能模块**：台账管理（查询、详情、编辑）、业务统计（出口/客户/商品/供应商）、系统设置
- **角色权限**：单证员、财务退税人员、管理员等多角色协同

优化建议覆盖：

- 必须修复的结构性问题（章节编号、关键定义缺失、逻辑矛盾）
- 内容补充建议（角色画像、术语、非功能性需求、版本管理）
- 一致性与命名规范（字段命名、状态值粒度、公式与字段分离）
- 各章节具体优化点
